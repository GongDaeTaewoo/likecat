# likecatProject
고양이 커뮤니티 프로젝트입니다.
# 기술스택
- Python
- HTML5
- Django
- Bootstrap5
# 기능
- 자유게시판,사진게시판(CRUD게시판)구현
- 마이페이지,비밀번호 변경,개인정보수정 구현
- 내부에 CRUD게시판이 있는 블로그 생성 구현
- 다른 사람의 블로그 검색,블로그 랭킹,블로그 추천 구현
